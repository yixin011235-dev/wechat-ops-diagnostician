# 诊断框架：律所号横纵分析法

本文是 skill 的判断逻辑本体。读完数据后，对照本文做诊断。

**横纵法**：纵＝这个号自己的时间趋势（数据给）；横＝它在行业竞争图谱里的位置与原型（知识给）。先用横定位"它该是什么、好的上限在哪",再用纵看"它自己走成了什么样",两者交汇出判断。

## 1. 第一原则：目标是输入，不是假设

一个号"好不好",完全取决于它自己的定位。本地案源号、全国思想领导号、招聘号、政企背书号——评判标准完全不同。**诊断第 0 步永远是确认这个号的原型和目标**，所有指标对着那个目标解读，绝不套一个通用目标（比如默认"阅读量越高越好"）。给襄阳本地服务号套上"全国触达"的目标，是典型的归因错误。

这也让 skill 可移植：换个号，喂它的原型和目标，同一套引擎重跑。产出永远是"对这个号的目标而言好不好",不是绝对跑分。

---

## 2. 横向：五类玩家原型（定位菜单）

先判这个号属于哪一类。每一类的真实角色、能力上限、"好"的定义都不同。**不要拿区域所号去对标天同。**

| 原型 | 代表 | 真实角色 | 主目标 | "好"长什么样 | 现实上限 |
|---|---|---|---|---|---|
| 红圈/大所官方号 | 金杜、盈科 | 品牌标配 + 雇主品牌 | 品牌背书、招聘 | 专业形象稳、招聘吸引力强 | To-B 不靠它获客，单篇数千–一两万 |
| 专业垂类 IP 号 | 天同诉讼圈、高杉LEGAL | 个人 IP + 垂直深耕 | 案源、培训/出版变现 | 粉丝粘性、付费转化 | 高度依赖创始人，精力难持续 |
| 法律新媒体平台 | 智合、律新社、新则 | 行业媒体 + B 端服务 | 榜单/培训/广告收入 | 行业话语权 | 独立性常被质疑 |
| **区域/中小所号** | 泰和泰、明炬、申浩 | **本地品牌 + 政企背书 + 招聘 + 私域终点** | 本地口碑、政企法顾采购背书、引才、把触达导入私域 | **本地存在感 + 私域线索** | **个体声量小是常态；核心困境是"没人写"** |
| 跨界新兴 | 法律咨询公司、短视频律师IP | 流量获客 → 私域沉淀 | 案源/带货 | 流量爆发 | 乱象多、被监管整顿 |

**区域/中小所号是大多数中小律所的归类**。它的读者往往不是"潜在当事人",而是**地方政府/国企法顾采购决策者**和**准备跳槽的律师**。这个原型不追全国流量，追的是本地圈层的可信存在感。诊断这类号时：
- 荣誉/获奖/对外交流/月例会这类"团队动态"，对它**是有用的**——服务政企背书 + 招聘。别一律斥为虚荣。
- 但也别把团队动态的高阅读当成案源成功——它触达的是已认识律所的人。
- 真正稀缺的是**持续产出能力**（"没人写"）和**把触达转成私域线索的出口**。

## 3. 目标分类法

一个号可能同时背几个目标，需让运营方按权重排序：

- **本地品牌**：在本地圈层维持可信存在感（区域所核心）
- **政企法顾背书**：让政府/国企采购决策者看到专业与活跃度
- **招聘引才**：让准备跳槽的律师觉得"值得来"
- **案源（C 端）**：本地个人/中小企业的具体法律需求
- **案源（B 端法顾）**：常年法律顾问类机构客户
- **思想领导/同行交流**：在专业人群里建立权威（垂类 IP 路线）

每个指标对哪个目标负责，不一样。比如：搜一搜阅读对"案源"和"思想领导"有意义，对"政企背书"几乎无关；荣誉稿对"政企背书/招聘"有意义，对"案源"无关。**诊断必须把指标挂到它真正服务的目标上**，而不是一锅烩。

## 4. 结构性现实校准（别把常态当失败）

这些是全行业公众号的结构性事实，不是某个号的毛病。诊断时不要把它们标成问题：

- **打开率低是常态**：2017 年起超 80% 的公众号打开率不足 5%，此后继续走低至不足 2%。推送触达小，不等于这个号失败。
- **信息流改版（2018）让品牌号天然吃亏**：订阅号只显示头条+次条、弱化账号强化内容。区域所这种"我是某所、今天发篇专业文"的品牌号，在 feed 分发里本就劣势。
- **流量在离开公众号**：主战场转向短视频/视频号 + 私域。最可能的未来（且已在发生）是——**公众号退化为"基础设施",像公司官网：必须有，但没人指望它直接带案源。**
- **内容同质化 + 灰色竞争**：43 万家法律咨询公司用低价/虚假承诺/冒充律师抢 C 端注意力，拉低公众对"律师公众号"的整体信任。

校准含义：对一个区域所号，**不要因为它推送打开率低、不爆款、规模小就判它失败**。这些是赛道常态。要判的是：它在自己原型的现实角色里，把该做的做到了没有。

## 5. 公域→私域：2026 年公众号的真实案源角色

当下主流的案源链路是：**公域获客（搜索/短视频/朋友圈）→ 加微信/企微（私域）→ 公众号深度内容沉淀信任 → 私域里转化咨询/案源。**

关键判断：**公众号本身不是转化器，私域才是。** 公众号的案源贡献＝两件事——
1. 做"深度内容存档站",让加了私域的人读到、建立信任；
2. 做"公域触达的私域入口",把搜索/本地来的陌生人导进企微。

所以对任何以案源为目标的号，诊断都会指向同一个杠杆：**有没有一条把读者导入私域的出口**。没有这条出口，再多触达对案源都是漏的——不是因为内容差，是因为链路断了。这也是为什么"正文内企微来源码"几乎总是案源类号的第一优先动作（见 `source_key_spec.md`）。

## 6. 指标 → 引擎 → 杠杆 映射

每个后台指标都是某个内容质量维度的代理变量。先按**来源**判这个号靠哪台引擎在跑：

| 来源占比高 | 说明的引擎 | 它在测 | 对应杠杆 |
|---|---|---|---|
| 搜一搜 | 搜索/常青引擎 | 选题需求、标题与搜索词匹配 | 问题式标题、常青选题、合集聚类 |
| 朋友圈 | 社交/熟人圈引擎 | 社交货币、本地关系网 | 立场、可转述性、本地相关 |
| 公众号消息 | 存量推送 | 标题、与存量粉丝信任 | 标题、推送时间（受打开率常态限制，杠杆有限）|
| 聊天会话 | 定向转发 | 对具体人有用 | 实用性、可转发 |
| 推荐(看一看) | 算法 | 广谱共鸣 + 在看率 | 多数小号算法几乎不响，别在此耗力 |

下游互动→质量映射：
- **完读率/时长**（需单篇导出）→ 开头、结构、节奏、长度
- **分享** → 社交货币
- **收藏** → 工具/可复用价值（对案源号，一次收藏＝一个潜在线索）
- **在看/留言**（本导出无）→ 认同/争议
- **阅读后关注、企微加好友、咨询**（需埋点）→ 作者价值证明 + 关注/加微理由 + CTA

诊断动作：先定位漏斗哪一层在漏（来源结构异常？完读低？无转化出口？），归因到结构原因，再开对应杠杆——**别看到"阅读低"就笼统改标题**。

## 7. 信噪纪律

- **小样本**：中小所号月阅读可能就几千，转化是稀有事件。单篇、单月波动噪声大——结论是"方向级",不是精确归因。
- **自己的基线**：绝对数字无意义。按来源、按内容类型归一化，跟这个号自己的滚动基线比；近期加权优于全期。
- **分母变化**：内容指标对着粉丝基数演化读。一波非主线爆文带来低意图关注后，后续每篇率值会"显得下滑",那是分母被稀释，不是内容变差。
- **漂移**：跨月看来源结构和受众结构的趋势，但按体量当趋势看、别过度解读小波动。
- 报结论前先问：这个差异在它自己基线的正常波动之外吗？有没有外部偶然（某大号转了一下、选题恰好时效）？

## 8. 合规视角

律师执业宣传受严格规制，诊断给的动作必须合规：

- **不得虚假/夸大**：禁"99% 胜诉率""保证胜诉"等承诺式表达（律师业务推广行为规则、各地律协自媒体公约）。
- **AI 代笔规范**：微信明确禁止 AI 代笔，三类行为封号。AI 在律所号里是**生产端提效工具**，不是内容端量产替代——产出仍需律师把关、署名真实。
- **个人信息**：留资/咨询数据遵个人信息保护法 + 执业保密义务，告知-同意、最小必要。
- **不冒充、不盗名**：与灰色"网推律所"划清界限本身也是品牌资产。

凡建议涉及 CTA、引流、AI 辅助、数据采集，附一句合规边界。

## 9. 盲区清单（每次诊断都要声明）

微信免费导出答不了的，明说，别用推断填：

- 没有**完读率/阅读时长**（要单篇内容分析导出）
- 没有**送达数** → 算不出打开率
- 没有**净增/取消/关注来源** → 拉新来源未知
- 常读画像**只是铁粉**，不是全部受众；城市**只到层级**，非省份
- **搜索读者的地域/身份**完全未知——而这常是案源引擎的人群
- 没有**转化数据**（要企微埋点）
- 互动只有分享/收藏，无在看/点赞/留言

把盲区作为一等输出。诚实划定"这套数据能下什么结论、不能下什么",是这个 skill 可信的全部来源。
