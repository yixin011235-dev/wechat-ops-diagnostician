# 复盘输出模板

诊断产出严格按本模板。它是"方向级、目标相关、盲区优先"的诊断报告，不是数据罗列。

```markdown
# [号名] · 运营诊断复盘

> **对象**：[账号] | **原型**：[五类之一] | **目标（评判标尺）**：[按权重列目标]
> **数据基础**：[导出类型 + 时间范围]

## 一、先划边界：能下什么、不能下什么
[账号体量一句话 + 信噪提示（小样本=方向级）]
[盲区清单：常读≠受众、城市只到层级、搜索地域未知、无完读率/送达/转化…]
一句话：下面是方向级结论，单篇/单月细节别当精确归因。

## 二、一句话诊断
[这个号是什么引擎组合、对着它的目标哪块成、哪块漏，一句话讲清]

## 三、分层体检（对着目标读）
### 3.1 触达：靠哪台引擎（来源结构 + 漂移）
### 3.2 受众：谁在反复读（标注是铁粉非全部；体量、城市层级、年龄、性别 + 利好/隐患）
### 3.3 内容组合：什么读得多、什么有真实目标价值（top 文 + 内容类型×来源 + 长尾）
### 3.4 转化：出口在不在（若无埋点，明确标为最大盲区/缺口）

## 四、对着目标：什么 working、什么在漏
[working/漏 对照表，每行挂到具体目标，按原型现实角色判，不与赛道常态较劲]

## 五、动作清单（按杠杆排序）
[P1…Pn，每条：动作 + 它修哪个漏 + 涉及 CTA/引流/AI/采集时附合规边界]
[把改法交棒给手艺 skill：标题→wechat-title-expert，结构→wechat-writing-coach，涨粉/私域→personal-brand-system]

## 六、重设"赢"的标准
[按原型重设成功定义，纠正"阅读量=成功"的误读；触达质量+转化 > 规模]
```

## 写作纪律
- **先边界后结论**：把盲区放最前，这是可信度的来源（"冷静经验主义翻译者"身份落点）。
- **每个判断挂目标**：不一锅烩，指标对应它真正服务的目标。
- **原型相对**：对区域所号别拿天同当尺；小体量、低打开率、不爆款是常态，不是失败。
- **动作要落到杠杆**：每条动作说清修哪个漏、可回测；案源类几乎总指向"私域出口"。
- **不重复手艺活**：本 skill 只做测量/归因/诊断，具体改法交给已有写作类 skill。
